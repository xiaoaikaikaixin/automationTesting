# Trading Slip Creation with Basis CD

## 1. Login and Navigation
- Login to the system
- Navigate to the Contract menu
- Select 'New Trading Slip' from Contract menu
- Wait for the Trading Slip - New record to load

## 2. Main tab details
- Click 'Division' dropdown list
- Select 'Fatty Alcohol' option
- Enter 'saleEqualNumber' in Sales Equal Number field
- Click 'Supplier' dropdown list
- Select 'Musim Mas, PT' option
- Verify that 'Factory Location' value equals to 'KIM II'
- Click 'Seller' dropdown list
- Select 'Inter-Continental Oils & Fats Pte. Ltd.' option
- Click Buyer pop up button and select the Buyer 'ICOF America Inc.' option
- Enter 'buyerPONo' in Buyer PO No. field
- Click 'Payment Term' dropdown list
- Select 'TT100_120daysBLDateInTrust' option
- Click 'Basis' dropdown list
- Select 'CIP' option
- Click 'Traded By' dropdown list
- Select 'Agus Salim' option
- Click 'Save' button
- Extract value from the 'Trading Slip No.' field
- Extract value from the 'Approval Status' field
- Verify that the Approval Status equals to 'Draft'
- Verify that the Trading Slip number is displayed correctly


## 3. Shipping tab details
- Click the 'Shipping' tab
- Click Delivery Mode dropdown list
- Select '20FT FCL' option
- Type '2' in 'Total No. of Units' field
- In 'Discharge Port's Country' field, fill in value 'United States of America' and hit Tab key
- Click 'Port of Discharge' dropdown 
- Click option 'Chicago'
- Click 'Override Budgeted Freight Cost' checkbox
- Type '500' in the 'Budgeted Freight Cost' field
- Click 'Override Freight Rate/MT' checkbox
- Type '300' in the 'Freight Rate/MT' field
- Select 'N/A' for Cargo Nature
- Click 'Save' button


## 4. Product tab details
- Click the 'Product Details' tab
- Click 'Add Item' button
- Click the Product Pop up button and select the Product 'Mascol 24 Lauryl Myristyl Alcohol'
- Click the 'RSPO SCCS Supply Chain Model' dropdown list and select option 'Mass Balance'
- Tick checkbox of 'GMP'
- Tick checkbox of 'NDPE'
- Tick checkbox of 'ISCC'
- Verify Buyer Goods Description shown the value 'GMP+ FSA Assured'
- Click the 'Product Specification' dropdown list and select option 'AL240-01 Rev. 3.00'
- Verify the Product Specification Details's value is 'AL240-01 Rev. 3.00'
- Click the 'Delivery UOM' dropdown list and select option 'MT'
- Type '100' in 'Quantity (UOM)'
- Type '1000' in 'Price/UOM' field
- Verify the Amount USD is equal to typed value 'Quantity (UOM)' * 'Price/UOM'
- Click the 'Packaging Profile Name' dropdown list and select option 'Flexibags - TL/BD'
- Click the 'Shipping Marking' dropdown list and select option 'Customized'
- Type 'Shipping Marking if Customize' in the Shipping Marking if Customize field
- Click the 'Packaging Marking' dropdown list and select option 'Customized'
- Type 'Packaging Marking if Customize' in the Packaging Marking if Customize field
- Click the 'Pallet Profile Name' dropdown list and select option 'Customized'
- Type 'Pallet Marking if Customize' in the Pallet Marking if Customize field
- Click the 'Insulation Type' dropdown list and select option '1 Layer Cartonboard'
- Verify the 'Product Remark' contains value 'GMP+ FSA Assured'
- Verify the 'Product Remark' contains value 'This is RSPO-MB + NDPE Product'
- Verify the 'Product Remark' contains value 'This is ISCC Trade'
- Click 'Save' button


## 5. Special Reqs. tab details
- Click the 'Special Reqs.' tab
- Click the 'Special Product Requirement' dropdown and select option 'Food Grade'
- Click the 'Prior Cargo Requirement' dropdown and select option 'Others'
- Type 'Others' in the 'Prior Cargo Requirement (Others)' field
- Type 'Testing ------------Additional Heating Requirement' in the 'Additional Heating Requirement' filed
- Type 'Testing SC remarks--------------------' in the 'SC Remarks' field
- Verify the 'Internal Remarks' contains value 'GMP+ FSA Assured'
- Verify the 'Internal Remarks' contains value 'This is RSPO-MB + NDPE Product'
- Verify the 'Internal Remarks' contains value 'This is ISCC Trade'
- Verify the 'Sanction Clause' contains value 'The Buyer confirms that the goods in this Sales Contract will not be directly or indirectly (i) sold to any party that is located in Cuba, Iran, Syria, North Korea, Crimea, Donetsk, Luhansk, Kherson, Zaporizhzhia and Sudan (collectively, the “Restricted Countries”), or (ii) shipped through the Restricted Countries, or (iii) sold to any individual or entity that is the subject of any sanction, prohibition or restriction imposed by any state, country, supranational or international governmental organisation or other authority (Collectively, “Sanctions”). If the Buyer is arranging shipment of the goods themselves, the Buyer also confirms that the goods in this Sales Contract will not be shipped directly or indirectly using (i) vessels that are registered in the Restricted Countries, (ii) vessel owners or shipping lines that are located in, or whose ultimate beneficial owners are located in the Restricted Countries, or (iii) vessels, vessel owners or shipping lines that are subject of Sanctions.'
