# Trading Slip Creation with Basis CD

## 1. Login and Navigation
- Login to the system
- Navigate to the Contract menu
- Select 'New Trading Slip' from Contract menu
- Wait for the Trading Slip - New record to load

## 2. Main tab details
- Click 'Division' dropdown list
- Select 'Fatty Alcohol' option
- Enter 'saleEqualNumber' in Sales Equal Number field
- Click 'Supplier' dropdown list
- Select 'Musim Mas, PT' option
- Verify that 'Factory Location' value equals to 'KIM II'
- Click 'Seller' dropdown list
- Select 'Inter-Continental Oils & Fats Pte. Ltd.' option
- Click Buyer pop up button and select the Buyer 'ICOF America Inc.' option
- Enter 'buyerPONo' in Buyer PO No. field
- Click 'Payment Term' dropdown list
- Select 'TT100_120daysBLDateInTrust' option
- Click 'Basis' dropdown list
- Select 'CIP' option
- Click 'Traded By' dropdown list
- Select 'Agus Salim' option
- Click 'Save' button
- Extract value from the 'Trading Slip No.' field
- Extract value from the 'Approval Status' field
- Verify that the Approval Status equals to 'Draft'
- Verify that the Trading Slip number is displayed correctly
